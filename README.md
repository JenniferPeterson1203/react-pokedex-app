# Jennifer's Pokédex

A futuristic neon-themed Pokédex built with React and PokéAPI featuring battle analysis, evolution tracking, favorites, animations, and responsive design.

---

## ✨ Features

- Pokémon search by name or number
- Favorites system with localStorage
- Evolution chain navigation
- Battle analyzer (weaknesses/resistances)
- Pokédex lore and habitat data
- Neon scan animations
- Responsive mobile and tablet layouts
- Dark and light mode

---

## 🛠 Built With

- React
- JavaScript
- CSS
- PokéAPI
- Vite

---

## 🚀 Installation Instructions

```bash
npm install
npm run dev
```
